# 📁 ПАПКА ДЛЯ ИЗОБРАЖЕНИЙ

## Структура:

```
images/
├── covers/           ← Обложки статей (главные изображения)
│   ├── health-1.webp
│   ├── health-2.webp
│   └── ...
└── articles/         ← Изображения внутри статей
    ├── diagrams/
    ├── photos/
    └── ...
```

## Требования:

✅ **Формат:** WebP (или JPG сжатый)
✅ **Размер:** 800x450 px (для обложек)
✅ **Вес:** < 100 KB
✅ **Имена:** только латиница, дефисы

## Быстрая оптимизация:

1. Открыть: https://squoosh.app/
2. Загрузить изображение
3. Настройки:
   - Format: WebP
   - Quality: 75
   - Resize: 800x450
4. Скачать и положить сюда

## Использование в статьях:

```markdown
# Обложка статьи (в frontmatter):
image: "/images/covers/health-nutrition.webp"

# Изображение внутри статьи:
![Описание](/images/photo.webp)
```
